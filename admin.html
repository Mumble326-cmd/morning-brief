How to update keywords:
  1. Edit the tags below — add or remove search terms per client
  2. Click Download keywords.json to save the updated file
  3. Go to github.com/Mumble326-cmd/morning-brief
  4. Click Add file → Upload files → drag in the downloaded file → Commit changes
  5. Go to Actions → Daily Morning Brief → Run workflow to refresh the brief immediately

To load an existing keywords.json from your computer: use Upload keywords.json above.
Changes only take effect after downloading the JSON, uploading to GitHub, and triggering the Action. The brief auto-runs every weekday at 6:30 AM SL time.
